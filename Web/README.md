# Archive·存迹 — SaaS 平台

Archive 的 Web 全栈应用平台，包含产品营销页和 Web 应用框架。

## 技术栈

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4 + Framer Motion + GSAP
- Lucide React（图标）

## 目录结构

```
src/app/
├── (marketing)/    # 产品官网页面（首页/功能/下载/更新日志/关于）
├── (app)/          # Web 应用路由（开发中）
└── (auth)/         # 认证路由（开发中）
```

## 开发

```bash
npm install
npm run dev
```
